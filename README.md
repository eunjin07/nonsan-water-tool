# 논산시 상수도 사용량 인정조정 판정 도구

엑셀(.xlsx) 검침 자료를 드래그앤드롭으로 업로드하면 사용량 인정·조정 판정을 도와주는 웹 도구입니다.

## 특징
- 서버 없음: 단일 `index.html` 정적 페이지
- 업로드한 엑셀 파일은 **브라우저 내에서만 처리**되며 외부 서버로 전송되지 않습니다 (SheetJS 클라이언트 파싱)
- 외부 의존성: Google Fonts, cdnjs의 SheetJS(xlsx 0.18.5) — 인터넷 연결 필요

## 배포
GitHub 저장소를 Vercel에 연결하면 자동 배포됩니다. `main` 브랜치에 푸시할 때마다 자동으로 재배포됩니다.

## 사용 방법
1. 배포된 사이트에 접속
2. 월별 검침 엑셀 파일(.xlsx)을 화면에 드래그앤드롭
3. 판정 결과 확인
